# Design Pattern
MVVM/Clean Architecture

# Framework Used
- Kotlin Multiplatform Mobile
- Ktor API
- Koin Dependency Injection
- AndroidX
- SwiftUI
- Jetpack Compose
- Kotlin Coroutines
- KMM module for handling business logic
- assertk
- Unit Tests Implementation
